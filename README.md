Senior Frontend Developer specializing in **React**, **TypeScript**, and modern web architecture. I build **scalable, data-driven UI systems** that bridge the gap between complex backend logic and intuitive user experiences.

My focus is on **product-first engineering**—treating the frontend not just as a display layer, but as a complete ecosystem where performance, UX clarity, and maintainability are non-negotiable. I architect component libraries, optimize critical rendering paths, and ensure every interaction feels instant and intentional.

**What drives me:** Turning ambiguous product requirements into clean, predictable systems. Solving real-world UX problems with code. Building interfaces that feel premium and reliable.
